Documentation

Content

Create a REST API for an application called “myFlix” that interacts with a database that stores data about different movies. In the next Achievement, you’ll build the client-side component of this same application using REACT. By the end of Achievement 3, you’ll have a complete web application (client-side and server-side) built using full-stack JavaScript technologies that you can showcase in your portfolio. The project will demonstrate your mastery of full-stack JavaScript development, including APIs, web server frameworks, databases, business logic, authentication, data security, and more. The complete tech stack you’ll master is known as the MERN (MongoDB, Express, React, and Node.js) stack.


API requirements

  • Be a Node.js and Express application;
  • Use REST architecture, with URL endpoints corresponding to the data operations listed above;
  • Use at least three middleware modules, such as the body-parser package for reading data from requests and morgan for logging;
  • Use a “package.json” file;
  • Be built using MongoDB;
  • Must be modeled with Mongoose;
  • Provide movie information in JSON format;
  • Be tested in Postman;
  • Include user authentication and authorization code;
  • Include data validation logic;
  • Meet data security regulations;
  • The source code must be deployed to a publicly accessible platform like GitHub;
  • Must be deployed to Heroku.
  • The JavaScript code must be error-free;
